PosixBox is a Emulator written in C++ primarily built for running on a x86 system.

So far it can run shell files and have a Linux-like terminal experience, were working on ELF application running on the Emulated System side.

PosixBox runs on 2 different sides which are the Emulated System Side, and the Emulator side.

The emulator side is like root, the Emulated System borrows resources from the Emulator side.
